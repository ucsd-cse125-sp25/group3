\doxysubsubsection{Detailed Description}
Include \texorpdfstring{$<$}{<}\doxylink{type__ptr_8hpp}{glm/gtc/type\+\_\+ptr.\+hpp}\texorpdfstring{$>$}{>} to use the features of this extension.

Handles the interaction between pointers and vector, matrix types.

This extension defines an overloaded function, \doxylink{group__gtc__type__ptr_gaada83809aefafd7c3264772e6b24b685}{glm\+::value\+\_\+ptr}. It returns a pointer to the memory layout of the object. Matrix types store their values in column-\/major order.

This is useful for uploading data to matrices or copying data to buffer objects.
