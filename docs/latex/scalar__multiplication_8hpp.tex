Include \texorpdfstring{$<$}{<}\doxylink{scalar__multiplication_8hpp}{glm/gtx/scalar\+\_\+multiplication.\+hpp}\texorpdfstring{$>$}{>} to use the features of this extension.

Enables scalar multiplication for all types

Since GLSL is very strict about types, the following (often used) combinations do not work\+: double \texorpdfstring{$\ast$}{*} vec4 int \texorpdfstring{$\ast$}{*} vec4 vec4 / int So we\textquotesingle{}ll fix that! Of course "{}float \texorpdfstring{$\ast$}{*} vec4"{} should remain the same (hence the enable\+\_\+if magic)
