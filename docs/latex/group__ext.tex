\doxysubsection{Detailed Description}
Additional features not specified by GLSL specification. 

EXT extensions are fully tested and documented.

Even if it\textquotesingle{}s highly unrecommended, it\textquotesingle{}s possible to include all the extensions at once by including \texorpdfstring{$<$}{<}\doxylink{ext_8hpp}{glm/ext.\+hpp}\texorpdfstring{$>$}{>}.
