\doxysubsubsection{Detailed Description}
Include \texorpdfstring{$<$}{<}\doxylink{range_8hpp}{glm/gtx/range.\+hpp}\texorpdfstring{$>$}{>} to use the features of this extension.

Defines begin and end for vectors and matrices. Useful for range-\/based for loop. The range is defined over the elements, not over columns or rows (e.\+g. \doxylink{group__core__matrix_gac54cbc572f1349a255fe476ca6a42a7b}{mat4} has 16 elements).
